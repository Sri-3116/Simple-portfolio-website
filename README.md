# Portfolio Website

This is a simple personal portfolio website created using HTML and basic CSS. The project includes a Home page and a Contact page, designed to showcase personal information and provide an easy way for visitors to get in touch.

## Features
- Responsive and clean user interface
- Home page with personal introduction
- Contact page for communication details
- Built using HTML and CSS fundamentals
- Beginner-friendly project structure
